# HousingEstate

A console application that might be used by the developer to manage housing estates.

When the program starts you will see the main menu. There you can perform operations such as printing info about housing estate, managing rooms rent and content, saving information about people to file, etc. There is also implemented time passing mechanism. If the room rent expires the tenant of this room will recieve a Tenant Letter, in case the rent expired more than 30 days ago all the tenant and all people are evicted and their items are utilised. There are a lot more features that you can play with, check them all in the Menu class.
